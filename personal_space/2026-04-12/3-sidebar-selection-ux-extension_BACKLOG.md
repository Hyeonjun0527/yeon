# 3-sidebar-selection-ux-extension BACKLOG

## 차수 1

### 작업내용

- 상담 기록 워크스페이스 사이드바 선택 UX를 파일 관리자처럼 확장한다.
- Ctrl/Cmd + 클릭 개별 토글 선택, Ctrl/Cmd + A 전체 선택, Esc 해제를 지원한다.
- 기존 Shift 범위 선택은 유지하되 드래그 선택도 안전하게 동작하도록 보강한다.
- 선택 개수와 사용 가능한 단축키를 sidebar 안에서 바로 보이게 한다.

### 논의 필요

- 없음. 사용자는 파일 관리자 같은 선택 UX 확장을 명시적으로 요청했다.

### 선택지

1. Shift 범위 선택만 유지한다.
2. Ctrl/Cmd 토글과 전체 선택까지 추가한다.
3. 2번에 더해 안전한 드래그 선택과 선택 힌트까지 포함한다.

### 추천

- 3번. 현재 우클릭 삭제 메뉴와 가장 자연스럽게 이어지고 파일 관리자 같은 사용감을 완성할 수 있다.
