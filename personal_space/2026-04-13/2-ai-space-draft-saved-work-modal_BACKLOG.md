## 차수 1

### 작업내용

- AI로 스페이스 초안 만들기 화면의 현재 정보 구조를 점검한다.
- 저장된 가져오기 작업 영역을 메인 화면에서 축소하고, 작은 버튼을 눌렀을 때 모달로 열리도록 바꾼다.
- 기존 이어서 보기/삭제/새로고침 동작은 유지하되 화면의 1차 위계를 새 파일 가져오기에 맞춘다.
- 변경 후 lint, typecheck, build로 검증한다.

### 논의 필요

- 저장된 작업 버튼의 정확한 라벨을 얼마나 축약할지.
- 모달에서 기존 리스트를 그대로 재사용할지, 더 요약된 형태로 줄일지.

### 선택지

1. 메인 화면 상단에 작은 텍스트 버튼만 두고 별도 모달에서 전체 리스트를 노출한다.
2. 메인 화면에 최근 1개만 남기고 전체보기를 모달로 옮긴다.
3. 저장된 작업을 완전히 숨기고 빈 상태 버튼 그룹 안으로 넣는다.

### 추천

- 1번. 메인 CTA 위계를 흐리지 않으면서도 기존 저장 작업 접근성을 유지할 수 있다.
