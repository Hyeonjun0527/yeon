# 26. Codex OMC Skill Import

## 차수 1

### 작업내용

- `oh-my-claudecode` 공개 저장소의 `skills/*` 디렉터리를 현재 저장소의 `.codex/skills` 아래로 가져온다.
- 기존 로컬 스킬과 충돌하지 않는지 확인하고, 필요한 경우 `.codex/skills/README.md`에 출처와 운영 방식을 정리한다.
- 설치 후 현재 저장소 기준으로 어떤 OMC 스킬이 추가됐는지 검증한다.

### 논의 필요

- 없음. 현재 목표는 로컬 Codex 스킬 인벤토리에 OMC 스킬을 노출시키는 것이다.

### 선택지

- 선택지 1: 전역 `CODEX_HOME/skills`에만 설치한다.
- 선택지 2: 현재 저장소의 `.codex/skills`에도 vendoring 해서 프로젝트 로컬 스킬로 유지한다.

### 추천

- 선택지 2. 이 저장소를 열었을 때 바로 같은 스킬 인벤토리를 재현할 수 있고, 레포 안에서 source와 출처를 함께 관리할 수 있다.
