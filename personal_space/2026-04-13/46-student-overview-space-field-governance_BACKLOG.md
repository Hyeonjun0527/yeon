# 학생 상세 기본 항목 스페이스 필드 승격 백로그

## 1차

### 작업내용

- 기본 섹션 항목을 스페이스별 field definition source of truth로 승격한다.
- 기본 overview 필드의 기본 세트를 스페이스 생성 시 함께 만들고, 기존 스페이스는 lazy init으로 보완한다.
- 필드 삭제는 내부적으로 소프트 딜리트로 전환한다.
- 학생 상세 개요에서 hover/right-click 메뉴, 이름변경 모달, 삭제 확인 흐름을 붙인다.

### 논의 필요

- 없음

### 선택지

- 선택지 1: 기본 항목을 계속 하드코딩하고 라벨/숨김만 별도 설정으로 덧입히기
- 선택지 2: 기본 항목을 실제 스페이스 field definition으로 승격하기
