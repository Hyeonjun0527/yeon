# 11. 학생관리 도움말 다페이지 가이드 BACKLOG

## 차수 1. 학생관리 도움말을 기능별 페이지로 재구성

- 작업내용
  - 학생관리 도움말 모달을 단일 설명 카드에서 다페이지형 가이드로 바꾼다.
  - 학생 목록, Google Sheets 연동, 리포트, 출석·과제 보드를 각 페이지로 분리한다.
- 논의 필요
  - 없음. 사용자가 설명 범위를 직접 지정함.
- 선택지
  - 선택지 A: 긴 단일 본문 유지
  - 선택지 B: 여러 페이지로 분리
- 추천
  - 선택지 B. 운영 기능이 많아 한 화면에 모두 쏟아 넣으면 읽히지 않고, 페이지 분할이 정보 위계를 더 분명하게 만든다.
- 사용자 방향
  - 학생관리 도움말을 여러 페이지로 구성

## 차수 2. 실제 기능 기준으로 문구 정교화

- 작업내용
  - 구글 연동이 무엇인지, 학생관리에서 무엇을 할 수 있는지, 리포트는 어디서 추출하는지, 출석·과제 보드는 어떻게 쓰는지 실제 코드 기준으로 안내한다.
- 논의 필요
  - 없음
- 선택지
  - 선택지 A: 포괄적이지만 추상적인 소개 문구
  - 선택지 B: 실제 버튼/화면 흐름 기준의 운영 문구
- 추천
  - 선택지 B. 도움말은 마케팅 문구보다 실제 조작 흐름을 줄여 주는 안내가 더 유용하다.
