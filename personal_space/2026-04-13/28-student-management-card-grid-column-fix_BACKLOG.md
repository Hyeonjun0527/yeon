# 28. Student Management Card Grid Column Fix

## 차수 1

### 작업내용

- 수강생 관리 카드 뷰가 1열로만 쌓이는 원인을 수정한다.
- 카드 열 수 계산 기준과 실제 grid 렌더링 기준을 하나로 맞춘다.
- 데스크톱에서는 `N x M` 카드 레이아웃이 안정적으로 유지되도록 responsive 계산을 보강한다.

### 논의 필요

- 없음. 현재 카드 뷰는 의도와 다르게 세로 1열로만 표시되고 있다.

### 선택지

- 선택지 1: CSS `auto-fill`만 유지하고 너비 측정만 보정한다.
- 선택지 2: 너비 측정 기준과 grid column 템플릿을 모두 명시적으로 통일한다.

### 추천

- 선택지 2. 가상 스크롤의 row slicing과 실제 렌더 열 수가 다시 어긋나지 않게 할 수 있다.
