# 9. 모바일 실행 / 전체 앱 기동 명령

## 차수 1

### 작업내용

- 루트에서 모바일 앱을 더 바로 실행할 수 있도록 실행 스크립트를 정리한다.
- 웹, 모바일, race-server를 한 번에 띄우는 루트 명령을 추가한다.
- 실제로 명령을 실행해 세 앱이 동시에 부팅되는지 확인한다.

### 논의 필요

- `dev:all`의 대상 앱 범위를 `apps/*` 전체로 고정할지, 이후 새 앱이 생길 때 수동 필터를 유지할지 결정이 필요하다.

### 선택지

- A. `turbo run dev --parallel --filter=...`로 현재 앱 3개만 명시적으로 묶는다.
- B. `pnpm -r --parallel`로 dev 스크립트가 있는 패키지를 전부 띄운다.

### 추천

- A. 현재 저장소에서 실제 애플리케이션은 `web`, `mobile`, `race-server`로 명확하고, package 레벨 dev 스크립트까지 따라 올라오는 것을 막을 수 있다.
