# 27. Home Space Create Student Management Redirect

## 차수 1

### 작업내용

- 상담관리 홈(`/home`)에서 스페이스 생성 후 학생관리 화면(`/home/student-management?spaceId=...`)으로 이동하도록 흐름을 수정한다.
- 첫 스페이스 생성 게이트와 홈 사이드바의 스페이스 생성 모달 모두 같은 후속 라우팅을 사용하도록 정리한다.
- 생성 직후 새 스페이스가 선택 상태로 보이도록 URL과 로컬 상태 전이를 맞춘다.

### 논의 필요

- 없음. 첫 계정 온보딩에서는 새 스페이스 생성 직후 학생관리 화면으로 진입하는 쪽이 더 자연스럽다.

### 선택지

- 선택지 1: 첫 스페이스 생성 게이트에서만 학생관리로 이동한다.
- 선택지 2: 상담관리 홈에서 스페이스를 만드는 모든 경로를 학생관리로 이동시킨다.

### 추천

- 선택지 2. 홈 화면의 생성 진입점이 둘로 나뉘어 있어도 결과 상태는 동일해야 사용자가 덜 혼란스럽다.
