## 차수 1

### 작업내용

- 홈 우측 AI 패널 인터넷 버튼을 실제 ON/OFF 토글로 연결한다.
- 웹검색 ON 상태를 채팅 요청 계약에 포함시키고, 대화창에 `웹검색중...` 진행 표시를 먼저 보이게 한다.
- 서버 채팅 경로에 OpenAI 웹검색 호출을 추가하고, 성공 시 출처 링크가 포함된 답변을 저장/표시한다.
- 웹검색 실패 시 일반 AI 답변으로 자동 fallback하는 경로를 추가한다.
- route/service 테스트를 추가하고 홈 패널 UI 회귀를 검증한다.

### 논의 필요

- 기존 `일반 모델 / 고급 모델` 토글은 이번 landing에서 서버 연동까지 확장할지 여부

### 선택지

- 인터넷 버튼만 실동작으로 고치고 모델 선택은 후속으로 남긴다.
- 인터넷 버튼과 모델 선택 둘 다 서버 연동까지 같이 확장한다.

### 추천

- 인터넷 버튼 실동작만 이번 landing에 포함한다. 현재 사용자가 막힌 지점은 웹검색 토글이며, 모델 선택까지 함께 확장하면 변경 단위가 커지고 테스트 범위가 불필요하게 넓어진다.
