# 공개 체크인 위치 검색 오류 피드백 백로그

## 1차

- 작업내용
  - Kakao Local API 설정 오류를 일반 502 대신 운영자가 이해할 수 있는 액션형 메시지로 변환한다.
  - 위치 검색 query의 React Query 재시도를 꺼서 실패 시 즉시 에러를 노출한다.
- 논의 필요
  - 운영자에게 Kakao 설정 경로까지 UI에 직접 노출할지
- 선택지
  - A. 서버에서 Kakao 오류를 그대로 노출
  - B. 서버에서 운영용 한국어 메시지로 번역
- 추천
  - B. 현재 사용자 문맥에서는 Kakao 원문 오류보다 설정 액션을 바로 알려주는 편이 맞다.
