# 홈 상담기록 테스트음성 빠른저장

## 1차

### 작업내용

- 홈의 `새 상담 기록` 모달 상단에 테스트 음성 3종 빠른 저장 버튼을 추가한다.
- 로컬 `voice-test-data`의 mp3 파일을 앱에서 읽을 수 있는 안전한 테스트용 route를 만든다.
- 버튼 클릭 시 별도 파일 선택 없이 기존 음성 업로드 흐름으로 바로 기록을 생성한다.

### 논의 필요

- 없음

### 선택지

- `public/test-data`로 파일을 복사해 정적 자산처럼 노출한다.
- 테스트용 route에서 허용된 파일만 매핑해 읽어 준다.

### 추천

- 테스트용 route에서 허용된 파일만 매핑해 읽어 준다.
