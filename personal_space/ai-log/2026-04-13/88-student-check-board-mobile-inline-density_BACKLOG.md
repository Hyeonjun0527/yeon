# 88. 학생 체크보드 모바일 인라인 밀도 조정

## 작업내용

- `현재 상태 빠른 수정` 블록을 좁은 화면에서 카드 스택 대신 한 줄 편집 중심 레이아웃으로 재구성한다.
- `이름/셀프체크/출석/과제/링크/저장` 정보 위계를 유지하되 모바일 세로 길이를 크게 줄인다.
- 기존 데스크톱/태블릿 레이아웃과 상태 저장 동작은 유지한다.

## 논의 필요

- 모바일에서 `최근 공개체크`를 본문 행 안에 유지할지, 보조 메타 1줄로 분리할지
- `셀프체크` 배지를 계속 노출할지, 너무 좁으면 이름 옆 축약 표시로 줄일지

## 선택지

1. 이름 행 + 입력 행 + 메타 행의 3단 초밀도 구성
2. 이름/배지/최근 공개체크를 한 행에 몰고 입력행을 1줄로 두는 2단 구성
3. 기존 카드형을 유지하되 패딩과 높이만 줄이는 소극 수정

## 추천

- 1번. 정보 손실 없이 높이를 가장 많이 줄이면서도 조작 실수를 줄이기 쉽다.
