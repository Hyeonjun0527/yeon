# 28-root-landing-kakao-login-modal_BACKLOG

## 작업내용

- 현재 `/landing`에 있는 랜딩 화면을 `/` 루트에서 바로 보이게 바꾼다.
- `시작하기` CTA를 누르면 로그인 모달이 즉시 열리도록 연결한다.
- 로그인 모달에는 카카오 로그인 버튼을 노출하고, 실제 인증 연동 전까지는 진입 UI와 문맥을 먼저 고정한다.

## 논의 필요

- `/landing` 경로를 유지할지, 루트로 리다이렉트할지

## 선택지

- A. `/`에 랜딩을 올리고 `/landing`은 `/`로 리다이렉트
- B. `/`와 `/landing` 둘 다 같은 랜딩을 렌더링

## 추천

- A. 사용자가 원하는 시작점은 루트 하나여야 한다. 중복 경로를 유지하면 진입 기준이 다시 흔들리므로 `/landing`은 `/`로 보내는 편이 명확하다.

## 사용자 방향

- 랜딩은 `/`에서 보이고, 시작하기를 누르면 카카오 로그인 버튼이 있는 로그인 모달이 바로 뜨게 구성
