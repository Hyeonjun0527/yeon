# 27-home-no-design-reset_BACKLOG

## 작업내용

- `http://localhost:3000/` 루트 화면에서 현재 워크스페이스 UI를 제거한다.
- 홈을 브라우저 기본 레이아웃에 가까운 평면 마크업으로 바꾸고, 전역 장식성 배경과 토큰 변수도 걷어낸다.
- 홈에서만 필요한 최소 문장과 리스트만 남기고 카드, 그라데이션, 장식용 타이포그래피를 제거한다.

## 논의 필요

- 로그인 폼 입력 필드까지 남길지, 안내 문장만 남길지

## 선택지

- A. 안내 문장과 다음 단계 리스트만 남기는 초평면 홈
- B. 로그인 입력 필드 2개는 남기되 스타일 없이 브라우저 기본 형태로 유지

## 추천

- B. 사용자가 말한 핵심 흐름이 로그인부터 시작한다는 점은 남겨야 하므로, 장식은 제거하되 로그인 입력 필드는 기본 형태로 유지하는 편이 의도 전달이 가장 명확하다.

## 사용자 방향

- localhost 홈에서 디자인을 전부 제거하고 최대한 기본 화면처럼 보이게 정리
