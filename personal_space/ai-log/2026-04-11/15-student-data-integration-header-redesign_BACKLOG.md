# 수강생 데이터 연동하기 헤더 리디자인 백로그

## 차수 1

### 작업내용

- `수강생 데이터 내보내기` 카드를 제거하고 `수강생 데이터 연동하기` 흐름으로 재구성한다.
- 학생관리 상단 헤더에 가져오기/내보내기/Google Sheets 연동을 한 묶음으로 배치한다.
- 가져오기와 내보내기를 모두 지원하고, 엑셀 수정 후 다시 가져오는 흐름까지 설명 가능한 구조로 만든다.
- 기존 `SheetExportPanel`의 기능은 유지하되 body 카드가 아니라 compact한 header action/inline panel 구조로 재배치한다.

### 논의 필요

- 없음. 사용자는 상단 통합 구조와 가져오기/내보내기 동시 지원을 명확히 요구했다.

### 선택지

1. 버튼 여러 개만 상단에 배치
2. 상단에 `수강생 데이터 연동하기` 섹션을 두고 inline 액션/연결 상태를 함께 제공

### 추천

- 2안. 단순 버튼 나열보다 현재 상태(Google Sheets 연결 여부)와 가져오기/내보내기 관계를 함께 보여줄 수 있어서 더 적합하다.
