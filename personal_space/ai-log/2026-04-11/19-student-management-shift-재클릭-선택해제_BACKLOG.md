# student-management shift 재클릭 선택해제 백로그

## 차수 1

### 작업내용

- `/home/student-management`에서 학생 다중 선택 로직을 확인한다.
- Shift를 누른 채 이미 선택된 학생을 다시 클릭했을 때 해당 학생 선택이 해제되도록 동작을 수정한다.
- 기존 단일 선택, 범위 선택, 우클릭 선택 흐름은 유지한다.

### 논의 필요

- Shift 클릭을 범위 선택 전용으로 볼지, 현재처럼 다중 선택 보조 토글까지 허용할지 UX 기준 정리가 필요하다.

### 선택지

1. Shift 클릭 시 이미 선택된 학생은 해제, 선택되지 않은 학생은 추가/범위 선택 유지
2. Shift 클릭은 항상 범위 선택만 수행하고 토글은 허용하지 않음

### 추천

- 선택지 1
- 현재 사용자가 기대하는 건 멀티 선택 상태를 더 자연스럽게 다루는 토글 동작이다. 기존 범위 선택 흐름을 깨지 않으면서 재클릭 해제를 추가하는 편이 안전하다.
