# 상담 기록 사이드바 우클릭 삭제 플로우 백로그

## 차수 1

### 작업내용

- 상담 기록 워크스페이스에서 파일 관리자처럼 우클릭 시 컨텍스트 메뉴를 띄운다.
- 삭제 대상은 상담기록, 수강생, 스페이스 3종으로 지원한다.
- 상담기록/스페이스는 기존 삭제 API를 연결하고, 수강생은 DELETE API를 새로 추가한다.
- 삭제 후 현재 선택 상태와 화면 표시가 stale state 없이 정리되도록 맞춘다.

### 논의 필요

- 없음. 사용자는 우클릭 기반 삭제 패턴을 명확히 요구했다.

### 선택지

1. 케밥 메뉴 버튼 추가
2. 우클릭 컨텍스트 메뉴 + 기존 기본 클릭 유지

### 추천

- 2안. 파일 관리자처럼 쓰고 싶다는 요구와 가장 직접적으로 맞는다.
