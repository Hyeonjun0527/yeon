## 차수 1

### 작업내용

- 다른 에이전트와 충돌 가능성이 있는 홈 retry UI 파일(`home/page.tsx`, `center-panel.tsx`)은 건드리지 않는다.
- `import-ocr-service.ts`에서 OCR 행/열 복원 시 빈 셀이 사라지지 않도록 표 복원 로직을 보강한다.
- `file-analysis-service.ts`에서 이미지 import fail-fast 규칙을 완화해 core 헤더만 있는 정상 표가 오탐 차단되지 않게 조정한다.
- 같은 범위에서 과도한 OCR fullText 프롬프트 길이를 줄일 수 있으면 함께 보정한다.
- 관련 서버 테스트를 추가/보강하고 검증한다.

### 논의 필요

- 이미지 표에서 `등록일`, `성별` 같은 컬럼을 고정 필드로 승격할지, 계속 custom field로 둘지는 별도 제품 결정이 필요하다.
- 현재 차수에서는 오탐 차단 완화와 OCR 구조 보존만 우선 처리한다.

### 선택지

1. OCR 행 복원만 수정하고 fail-fast는 그대로 둔다.
2. OCR 행 복원 + fail-fast 조건 완화 + 테스트 보강을 한 번에 처리한다.
