## 1차

### 작업내용
- 홈 페이지 로그인 진입 시 세션이 즉시 정리되는 원인을 확인한다.
- 정상 세션은 유지하고, 깨진 세션만 cleanup 라우트로 보내도록 조건을 수정한다.
- 브라우저에서 실제 로그인 세션 쿠키가 유지되는지 검증한다.

### 논의 필요
- 홈 페이지에서 로그인 후 기본 도착지를 계속 랜딩으로 둘지, 워크스페이스로 보낼지

### 선택지
- 선택지 1: 정상 세션도 무조건 cleanup
- 선택지 2: 정상 세션은 유지, 무효 세션만 cleanup
