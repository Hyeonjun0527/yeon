# 77. 분석 결과 내보내기 (Analysis Export)

## 예상 차수: 3차

## 현황

- AI 분석 결과는 클라이언트 메모리에만 존재 (새로고침 시 소멸)
- 기존 원문 클립보드 복사(`handleExportClipboard`)는 원문만 복사, AI 분석은 미포함
- 교사가 AI 분석을 보호자 전달, 학교 기록부, 보고서에 활용하려면 내보내기가 필수

## 목표

- AI 대화 내용을 클립보드/텍스트 파일/PDF로 내보내기
- 원문 + AI 분석을 결합한 종합 보고서 형태도 지원
- 내보내기 포맷이 바로 실무에 붙여넣을 수 있는 수준

---

### 차수 1: AI 대화 클립보드 복사 + 텍스트 다운로드

**작업내용**

- AI 패널 헤더에 "내보내기" 버튼 추가
- 클릭 시 드롭다운: "클립보드 복사" / "텍스트 파일 다운로드"
- 내보내기 내용 구성:

  ```
  [상담 분석 보고서]
  학생: {studentName}
  상담 제목: {sessionTitle}
  일시: {createdAt}

  --- AI 분석 ---
  {assistant 메시지들을 순서대로}

  --- 원문 요약 ---
  세그먼트 {count}개 · {textLength}자
  ```

- 텍스트 파일은 `Blob` + `URL.createObjectURL`로 다운로드 트리거

**논의 필요**: 없음. 클라이언트 단독 기능.

**추천**: 서버 호출 불필요, 즉시 구현 가능.

### 차수 2: 종합 보고서 내보내기 (원문 + AI 분석 통합)

**작업내용**

- 내보내기 드롭다운에 "종합 보고서" 옵션 추가
- 내용 구성:

  ```
  [상담 기록 종합 보고서]

  ## 기본 정보
  학생: {studentName} | 상담 유형: {counselingType}
  상담 제목: {sessionTitle}
  일시: {createdAt}

  ## AI 분석
  {assistant 메시지 마크다운 그대로}

  ## 상담 원문
  [00:00] 교사: ...
  [00:15] 학생: ...
  ...
  ```

- 마크다운 형식으로 복사 (노션, 구글 독스에 바로 붙여넣기 가능)
- 텍스트 파일 다운로드도 지원

**논의 필요**: 없음

**추천**: 마크다운이 범용성 최고. PDF는 다음 차수.

### 차수 3: PDF 내보내기

**작업내용**

- 클라이언트 사이드 PDF 생성 (서버 부하 없음)
- `@react-pdf/renderer` 또는 `html2canvas` + `jspdf` 사용
- 보고서 양식:
  - 상단: 학교명(설정 가능), 상담 기록 제목, 일시
  - 본문: AI 분석 요약 → 원문 전사 (페이지 나눔)
  - 하단: 생성 일시, 페이지 번호
- 한글 폰트 임베딩 필요 (Pretendard 또는 Noto Sans KR)

**논의 필요**

- PDF 라이브러리 선택

**선택지**

- A) `@react-pdf/renderer` — React 컴포넌트로 PDF 작성, 한글 지원 양호
- B) `jspdf` + `html2canvas` — HTML을 이미지로 변환 후 PDF, 레이아웃 유지 쉬움
- C) 서버 사이드 PDF (puppeteer/playwright) — 품질 최고, 서버 부하 큼

**추천**: A) `@react-pdf/renderer`. 클라이언트에서 완결, 한글 지원 검증됨.
