# Product Backlog

이 폴더는 실행 항목으로 관리해야 하는 개발 백로그의 공식 위치입니다.

## 원칙

- 백로그는 AI 작업 로그가 아니라 실행해야 할 일입니다.
- 새 백로그는 가능하면 주제별 문서로 정리합니다.
- 과거 날짜별 백로그는 `history/YYYY-MM-DD/`에 보관합니다.
- 완료/폐기 여부가 결정되면 문서 안에 상태를 표시하거나 별도 정리 문서로 승격합니다.
