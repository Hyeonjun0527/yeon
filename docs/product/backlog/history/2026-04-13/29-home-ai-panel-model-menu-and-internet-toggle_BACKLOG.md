## 차수 1

### 작업내용

- 상담 기록 홈 우측 AI 패널 하단 입력 영역의 모델 드롭다운이 패널에 잘리는 레이어 문제를 수정한다.
- 입력 툴바의 인터넷 버튼이 항상 disabled인 문제를 고쳐, 클릭 가능한 토글 컨트롤로 정리한다.
- 하단 입력 박스의 레이어/overflow 구조를 점검해 드롭다운과 토글 버튼이 작은 폭에서도 정상 동작하게 만든다.

### 논의 필요

- 인터넷 토글의 실제 검색 연동은 아직 없음

### 선택지

- 인터넷 버튼을 숨기거나 제거한다.
- 인터넷 버튼은 UI 토글로 살리고, 실제 검색 연동은 후속 작업으로 분리한다.

### 추천

- UI 토글로 먼저 살린다. 현재 문제는 "죽은 버튼처럼 보이는 상태" 자체가 UX 결함이므로, 상호작용 가능 상태를 먼저 회복하고 실제 검색 연동은 별도 landing 단위로 분리하는 편이 안전하다.
