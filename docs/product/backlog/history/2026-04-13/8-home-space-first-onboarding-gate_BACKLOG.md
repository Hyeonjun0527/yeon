## 차수 1

### 작업내용

- `/home` 진입 시 스페이스가 하나도 없으면 상담 워크스페이스 대신 스페이스 생성 온보딩 게이트를 먼저 보여준다.
- 온보딩 게이트에서 `파일 가져와 스페이스 만들기`와 `빈 스페이스 만들기` 두 CTA를 제공한다.
- 기존 `CreateSpaceModal`을 초기 스텝 지정 가능하게 확장해 게이트 CTA와 바로 연결한다.
- 기존 스페이스가 생기면 기존 상담 워크스페이스 흐름으로 자연스럽게 복귀한다.

### 논의 필요

- 스페이스가 이미 있어도 미선택 상태에서는 같은 게이트를 보여줄지 여부.
- 첫 진입 튜토리얼을 스페이스 우선 흐름으로 재구성할지 여부.

### 선택지

1. 스페이스가 0개일 때만 게이트를 보여준다.
2. currentSpace 미선택이면 항상 게이트를 보여준다.
3. 기존 empty state 문구만 바꾼다.

### 추천

- 현재 hook의 자동 선택 구조를 크게 흔들지 않도록 1번부터 적용하고, 필요 시 2번으로 확장한다.
