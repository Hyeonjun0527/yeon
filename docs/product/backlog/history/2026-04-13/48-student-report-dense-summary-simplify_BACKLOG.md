# 48-student-report-dense-summary-simplify

## 작업내용

- 수강생 상세 리포트 탭을 여러 상담 기록을 묶는 촘촘한 요약 리포트 구조로 단순화한다.
- 리포트 UI와 문서 생성기에서 `강조 메모`, `후속 액션`, `운영 메모`, `AI 분석 반영` 관련 요소를 제거한다.
- 리포트 본문을 `핵심 요약 + 상담 기록 요약`의 2단 구조로 고정한다.
- 사용자 대상 제품 설명성 문구를 제거하고, 상담 내용/마무리 상태 중심의 요약으로 재구성한다.

## 논의 필요

- 없음

## 선택지

1. UI에서만 항목을 숨기고 문서 생성기는 기존 구조를 유지한다.
2. `report-builder.ts`를 source of truth로 줄이고, UI도 그 결과에 맞춰 함께 단순화한다.

## 추천

- 2번
- 이유: 제거 대상이 UI 문구 수준이 아니라 문서 summary/meta/sections까지 걸쳐 있어서, 생성기와 화면을 같이 줄여야 의미가 일관된다.
