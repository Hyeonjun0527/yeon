# 수강생 상세 탭 컨텍스트 액션

## 작업내용

- 수강생 상세 탭 바에서 커스텀 탭만 데스크톱 우클릭, 모바일 롱프레스로 이름 변경/삭제 메뉴를 열 수 있게 만든다.
- `개요`, `상담기록`, `메모`, `리포트` 기본 탭은 이름 변경/삭제를 서버와 UI 양쪽에서 금지한다.
- 스페이스 설정에서는 기본 4개 탭을 숨기지 않고 읽기 전용 배지로만 표시하며, 편집/삭제/이동 UI를 제거한다.
- 커스텀 탭 삭제는 확인 모달을 거친다.

## 논의 필요

- 없음

## 선택지

- 시스템 탭을 스페이스 설정에서 숨긴다
- 시스템 탭을 읽기 전용 배지로 남긴다
- 커스텀 탭 삭제를 즉시 실행한다
- 커스텀 탭 삭제 전에 확인 모달을 띄운다

## 추천

- 시스템 탭은 읽기 전용 배지로 남긴다.
- 커스텀 탭 삭제는 확인 모달 후 실행한다.

## 사용자 방향

- 시스템 탭은 읽기 전용 배지로만 표시
- 커스텀 탭 삭제는 `정말 삭제할까요?` 확인 모달
