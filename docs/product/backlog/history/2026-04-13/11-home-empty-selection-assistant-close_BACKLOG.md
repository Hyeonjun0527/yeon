## 1차

### 작업내용

- 홈 상담기록 화면에서 선택된 기록이 없을 때 AI 어시스턴트 패널이 자동으로 닫히도록 상태 전이를 정리한다.
- 기존 기록 선택 시 열림/닫힘 UX를 깨지 않도록 선택 상태와 패널 상태의 source of truth를 확인하고 최소 수정으로 반영한다.
- 변경 범위에 맞는 테스트 또는 검증을 추가한다.

### 논의 필요

- 없음

### 선택지

- 선택된 기록이 없을 때 렌더링만 숨기고 열림 상태는 유지한다.
- 선택된 기록이 없을 때 패널 열림 상태 자체를 강제로 `false`로 동기화한다.

### 추천

- 선택된 기록이 없을 때 패널 열림 상태 자체를 `false`로 맞춘다. 선택 해제 이후에도 사이드 패널이 열린 상태값이 남지 않아 상태 정합성이 가장 좋다.
