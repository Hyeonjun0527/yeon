# 33. 공개 체크인 세션 과제 전용 모드 노출

## 작업내용

- 출석·과제 보드의 공개 체크인 세션 생성 폼에서 `과제만` 선택지를 노출한다.
- 세션 생성 카드의 보조 문구가 현재 선택한 체크 모드와 맞게 바뀌도록 정리한다.
- 기존 `출석 + 과제`, `출석만` 동작은 그대로 유지한다.

## 논의 필요

- 세션 제목도 체크 모드에 맞춰 자동으로 바꿀지 여부

## 선택지

- 선택지 A: select 옵션만 추가한다.
- 선택지 B: 옵션 추가와 함께 보조 문구/라벨도 현재 모드에 맞춰 동기화한다.

## 추천

- 선택지 B
