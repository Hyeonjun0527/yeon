# 공개 체크인 카카오 위치 검색 복귀 백로그

## 1차

### 작업내용

- 공개 체크인 위치 검색 공급자를 `Juso API`에서 `Kakao Local API`로 되돌린다.
- 서버 위치 검색 서비스와 테스트를 카카오 응답 기준으로 다시 맞춘다.
- `JUSO_API_CONFIRM_KEY`, `proj4` 등 Juso 전용 의존성과 설정 예시를 제거한다.
- 기존 학생 GPS 반경 검증 계약은 유지한다.

### 논의 필요

- 카카오 키는 이미 심사 완료 상태로 가정하고, 추가 승인 플로우는 코드 범위에서 다루지 않는다.

### 선택지

- 선택지 A: Juso/Kakao 이중 지원으로 공급자 선택 분기를 둔다.
- 선택지 B: Juso 구현을 제거하고 카카오 한 가지로 단순화한다.

### 추천

- 선택지 B
- 오늘 심사용 결과물 기준으로 실패 원인을 줄이려면 공급자를 하나로 단순화하는 편이 맞다.
