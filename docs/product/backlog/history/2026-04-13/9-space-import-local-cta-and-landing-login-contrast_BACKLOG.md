## 차수 1

### 작업내용

- 스페이스 가져오기 화면에서 `내 컴퓨터` 업로드 CTA를 더 강하게 보이도록 재배치한다.
- 테스트용 엑셀 파일을 바로 받을 수 있는 `테스트 데이터 다운로드` 버튼을 눈에 띄게 노출한다.
- 랜딩의 `로그인하고 시작하기` 버튼 대비와 클릭 가능성을 높인다.
- 변경 후 타입체크, 린트, 빌드, 브라우저 확인으로 회귀를 검증한다.

### 논의 필요

- 테스트 데이터 다운로드 CTA를 스페이스 가져오기 화면에만 둘지, 홈 게이트에도 둘지.
- 로그인 CTA를 단일 스타일로 유지할지, 상단/하단 섹션마다 다르게 둘지.

### 선택지

1. 기존 위치 유지 + 스타일만 강조한다.
2. 업로드/테스트 데이터를 별도 액션 블록으로 승격한다.
3. 테스트 데이터 버튼만 추가한다.

### 추천

- 2번처럼 업로드와 테스트 데이터를 같은 액션 블록으로 묶고, 로그인 CTA는 밝은 대비로 재조정하는 것이 가장 빠르고 명확하다.
