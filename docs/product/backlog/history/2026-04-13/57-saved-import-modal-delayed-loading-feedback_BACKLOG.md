# 저장된 가져오기 작업 모달 지연 로딩 피드백

## 작업내용

- 저장된 가져오기 작업 모달의 `새로고침`과 `삭제` 버튼 로딩 문구를 즉시 노출하지 않고, 요청이 `300ms`를 넘길 때만 `...` 형태로 보이게 조정한다.
- 빠르게 끝나는 요청에서는 버튼 문구를 유지해 flicker처럼 보이는 현상을 줄인다.
- 기존 모달 구조와 카드 디자인은 유지한다.

## 논의 필요

- 없음

## 선택지

- 즉시 로딩 문구 노출
- `300ms` 이상 지연 시에만 정적 `...` 문구 노출
- 점 개수 애니메이션 유지

## 추천

- `300ms` 이상 지연 시에만 정적 `...` 문구를 노출한다.

## 사용자 방향

- `300ms` 이상 지연 시에만 `새로고침 중...` / `삭제 중...` 노출
