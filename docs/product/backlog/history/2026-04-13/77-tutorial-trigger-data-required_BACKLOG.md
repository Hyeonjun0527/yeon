# 튜토리얼 버튼 데이터 의존 노출 BACKLOG

## 1차

### 작업내용

- 상단 `튜토리얼` 버튼의 노출 조건을 페이지별 필수 데이터/타겟 기준으로 재정의한다.
- 빈 상태나 핵심 데이터가 없는 상태에서는 튜토리얼 버튼이 보이지 않도록 수정한다.
- 상담 기록, 수강생 관리, 출석보드 각각의 필수 튜토리얼 타겟을 코드에 명시해 노출 기준을 일관되게 유지한다.

### 논의 필요

- 빈 상태 전용 튜토리얼을 이후 별도 가이드로 유지할지, 완전히 제거할지

### 선택지

- A. 빈 상태에서는 버튼을 숨기고, 실제 사용 가능한 데이터가 있을 때만 튜토리얼을 노출한다.
- B. 빈 상태 전용 튜토리얼을 유지하고 버튼도 계속 보여준다.

### 추천

- A. 실제 정보가 없을 때 튜토리얼이 오작동하거나 가치가 떨어지므로 버튼 자체를 숨기는 편이 맞다.

### 사용자 방향

- 정보가 없으면 튜토리얼 버튼을 보여주지 않는다.
