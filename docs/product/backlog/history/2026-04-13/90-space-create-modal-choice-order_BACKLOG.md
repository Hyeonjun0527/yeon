# 90-space-create-modal-choice-order_BACKLOG

## 차수 1

### 작업내용

- 스페이스 만들기 선택 모달에서 `AI로 파일 가져와 스페이스 만들기` CTA를 최상단으로 올린다.
- `빈 스페이스 만들기` CTA는 두 번째로 내려 기존 시각 스타일은 유지한 채 우선순위만 조정한다.
- 선택지 순서를 한 곳의 source of truth로 정리하고, 순서 회귀를 막는 최소 테스트를 추가한다.

### 논의 필요

- 없음

### 선택지

1. JSX에서 버튼 순서만 직접 바꾼다.
2. 선택지 메타데이터를 상수로 정리하고, 렌더 순서와 테스트가 같은 source of truth를 보게 만든다.

### 추천

- 2. UI 우선순위가 다시 뒤집히지 않도록 선택지 순서를 상수로 관리한다.
