# Team Docs

이 폴더는 팀원이 반복해서 참고하는 **공식 문서**를 관리하는 공간입니다.

## 문서 배치 기준

- 시스템 구조/경계: `docs/architecture/`
- 제품/기획/백로그: `docs/product/`
- SEO/검색 노출: `docs/seo/`
- 특정 프로젝트 문서: `docs/projects/{project-name}/`
- 개발/운영 가이드: `docs/guides/`
- 배포/인프라: `docs/deployment/`
- 공모전/제출 자료: `docs/contest/`
- 오래된 참고 자료: `docs/archive/`

## 원칙

1. 개인 이름 기준 폴더를 만들지 않습니다.
2. 날짜별 작업 로그는 공식 문서로 올리지 않습니다.
3. 팀원이 반복해서 참고할 결정된 내용만 공식 위치에 둡니다.
4. AI 협업 과정 기록은 루트 `ai-log/`에서 관리합니다.
5. 결정된 내용은 `docs/`로 승격하고, 과정 기록에는 링크만 남깁니다.
