## 솔루션 한 줄 요약

`YEON`은 운영 조직 소속 부트캠프 교강사가 수업 전 30분 안에 오늘 챙길 학생을 정리하고 개입까지 이어가게 만드는 AI 학생관리 CRM이다.

## 현재 기획 초점

- 3차에서는 교강사의 하루를 `수업 전`, `수업 중`, `수업 후`로 나누고, MVP가 먼저 해결할 순간을 `수업 전 준비 시간`으로 고정한다.
- 4차에서는 위험 신호를 `참여 저하`, `과제 지연`, `질문 없음`, `상담 메모 경고` 순으로 우선순위화한다.
- 5차에서는 제출용 제품 표현을 `교강사용 AI 학생관리 CRM`으로 확정하고, 한 줄 설명과 문제 정의 문구를 함께 고정한다.
