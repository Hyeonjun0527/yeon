# 홈 스페이스 가져오기 완료 후 생성 스페이스 네비게이션 백로그

## 차수 1

### 작업내용

- 상담관리 첫 온보딩에서 스페이스 생성 모달의 AI 가져오기 완료 후 현재 화면에 머무는 원인을 확인한다.
- 가져오기 완료 응답의 `spaceIds`를 소비해 생성된 스페이스로 이동하도록 홈 모달 경로를 수정한다.
- 기존 학생관리 화면 등 다른 가져오기 완료 흐름과 충돌하지 않는지 확인한다.

### 논의 필요

- 여러 스페이스가 한 번에 생성될 때 어느 스페이스로 이동할지 기준을 정해야 한다.

### 선택지

- 첫 번째 생성 스페이스로 이동한다.
- 별도 선택 UI를 띄운다.
- 기존 화면에 남기고 토스트만 띄운다.

### 추천

- 현재 응답의 `spaceIds[0]`로 즉시 이동한다. 사용자는 방금 만든 스페이스를 이어서 확인하려는 흐름이므로 추가 선택 단계보다 자연스럽다.
