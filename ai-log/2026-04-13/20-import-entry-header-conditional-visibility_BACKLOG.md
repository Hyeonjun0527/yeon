## 차수 1

### 작업내용

- AI 가져오기 진입 상태에서는 설명과 3개 CTA를 다시 보이게 복구한다.
- 대신 파일을 이미 선택해 분석/표 검토 단계로 들어간 상태에서만 해당 헤더가 숨겨지도록 조건을 현재 상태 기준으로 되돌린다.
- 학생관리 스페이스 import 모달이 `CloudImportInline`의 상태 기반 헤더 제어를 덮어쓰지 않게 정리한다.

### 논의 필요

- 없음

### 선택지

- 상단 안내 헤더를 항상 숨긴다.
- 파일 선택 전에는 보이고, 분석 워크스페이스 상태에서만 숨긴다.

### 추천

- 상태 기반으로만 숨긴다. 초기 진입 단계에서는 CTA와 설명이 실제 행동 유도에 필요하고, 표 검토 단계에서만 공간 절약이 중요하다.
