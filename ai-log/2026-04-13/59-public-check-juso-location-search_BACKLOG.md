# 공개 체크인 Juso 위치 검색 전환 백로그

## 1차

- 작업내용
  - `Kakao Local API` 기반 공개 체크인 위치 검색을 `Juso 검색 API + 좌표 API` 조합으로 교체한다.
  - Juso 좌표 `entX/entY`를 학생 GPS 비교에 사용할 `WGS84 위경도`로 변환한다.
  - 기존 공개 체크인 UI와 GPS 반경 검증 구조는 유지한다.
  - env 예시, 테스트, 오류 메시지를 Juso 기준으로 정리한다.
- 논의 필요
  - 좌표 변환을 외부 라이브러리로 처리할지 직접 구현할지
- 선택지
  - A. `proj4`로 UTM-K(GRS80) -> WGS84 변환
  - B. 직접 좌표 변환 수식 구현
- 추천
  - A. 오늘 심사용 결과물 기준으로 가장 빠르고 안정적이다.
