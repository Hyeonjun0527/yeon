# 89. 레거시 비상연락 필드 링크 렌더 보정

## 작업내용

- `비상 연락 정보` 섹션에서 `긴급연락처`, `비상연락처`, `긴급연락처관계`, `비상연락처관계` 같은 레거시 guardian 필드가 링크처럼 보이지 않도록 평문 렌더링으로 보정한다.
- 커스텀 필드 타입 추론에서 `...관계` 계열 이름이 `phone`으로 오분류되지 않게 규칙을 보강한다.
- 기존 편집 흐름과 저장 API 계약은 유지한다.

## 논의 필요

- 기존 DB에 이미 잘못 저장된 `fieldType=phone` 값을 마이그레이션으로 바로 고칠지, 우선 UI 보정만 둘지

## 선택지

1. UI 렌더 보정 + 추론 규칙 수정
2. DB fieldType 일괄 마이그레이션 + UI 수정 없음
3. 추론 규칙만 수정하고 기존 데이터는 방치

## 추천

- 1번. 현재 사용자 화면 문제를 즉시 해결하면서 앞으로 같은 오분류도 막을 수 있다.
