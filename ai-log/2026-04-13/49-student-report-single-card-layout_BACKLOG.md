# 49-student-report-single-card-layout

## 작업내용

- 수강생 상세 리포트 탭의 외곽 카드를 하나로 합친다.
- 현재의 `설정 카드 + 결과 카드` 2단 구조를 제거하고, 하나의 리포트 워크스페이스 카드 안에서 헤더, 범위 선택, 메타, 본문이 이어지도록 재배치한다.
- 리포트 종이 아이콘은 유지하고, 내부 구획은 divider와 가벼운 정보 구역 정도로만 남긴다.

## 논의 필요

- 없음

## 선택지

1. 카드 두 개를 유지한 채 간격만 줄인다.
2. 외곽 카드는 하나로 합치고, 내부는 평평한 구획으로만 나눈다.

## 추천

- 2번
- 이유: 현재 문제는 정보량보다 카드 위계가 둘로 갈라져 보이는 점이다. 외곽 카드를 하나로 줄여야 리포트 워크스페이스처럼 읽힌다.
