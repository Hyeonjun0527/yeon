# 홈 인사이트 배너 전역 dismiss 백로그

## 차수 1

### 작업내용

- 홈 인사이트 배너를 `상담 이력 없음`, `상담 간격 주의` 두 개로 분리한다.
- 각 배너에 X 버튼을 추가하고, 클릭 시 계정 전역 기준으로 3시간 숨김 상태를 저장한다.
- 숨김 상태 조회/저장을 위한 서버 persistence, 서비스, route handler, API 계약을 추가한다.
- 홈 배너 UI를 TanStack Query 기반 조회/Mutation으로 연결한다.

### 논의 필요

- 없음

### 선택지

- 사용자 테이블에 preference 컬럼을 추가한다.
- 전용 dismiss 테이블을 추가한다.

### 추천

- 전용 dismiss 테이블을 추가한다.
  - 이유: 현재 `users` 스키마를 덜 건드리고, 배너 종류별 TTL 상태를 명확하게 upsert/조회할 수 있다.
