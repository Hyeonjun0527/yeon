## 1차

### 작업내용

- 학생 상세의 커스텀 탭 본문이 비어 있을 때 중앙 empty state에 `커스텀 필드 추가` 버튼을 노출한다.
- 버튼 클릭 시 기존 `AddCustomFieldModal`을 현재 커스텀 탭 기준으로 연다.
- 필드가 생성되면 현재 탭 본문에 즉시 반영되고, 빈 상태 CTA는 사라지게 유지한다.

### 논의 필요

- 없음

### 선택지

- 기존 개요용 필드 추가 모달 상태를 탭 ID 기반 공통 상태로 일반화
- 커스텀 탭 전용 모달 상태를 별도로 추가

### 추천

- 기존 모달을 재사용하면서 탭 ID 기반 공통 상태로 일반화하는 쪽이 상태 중복이 적고 회귀 위험이 낮다.
