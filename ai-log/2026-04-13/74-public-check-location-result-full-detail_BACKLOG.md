# 공개 체크인 위치 검색 결과 상세 노출 백로그

## 차수 1

### 작업내용

- 공개 체크인 위치 검색 결과 UI를 현재 2줄 truncate 구조에서 `풀 노출` 구조로 확장한다.
- 결과 카드에 장소명, 도로명 주소, 지번 주소를 각각 읽을 수 있게 표시한다.
- 선택된 위치 요약 박스도 같은 정보 구조를 유지해 사용자가 생성 직전 다시 확인할 수 있게 맞춘다.

### 논의 필요

- 우편번호를 모든 결과에 채우기 위해 추가 조회를 강제할지 여부.

### 선택지

- 추가 조회 없이 현재 응답 필드만 풀 노출한다.
- 우편번호를 위해 추가 조회를 넣는다.

### 추천

- 추가 조회 없이 현재 응답 필드만 풀 노출한다. 이번 목적은 선택 정확도 개선이고, 응답 속도를 해치지 않는 선에서 도로명/지번 정보를 명확히 보여주는 것이 우선이다.
