## 1차

### 작업내용

- 기존 대용량 테스트 데이터 다운로드를 유지한다.
- `1기~3기`, 기수당 `백엔드/프론트` 2트랙, 트랙당 약 15명 규모의 경량 테스트 데이터 xlsx를 추가한다.
- 스페이스 생성 모달과 가져오기 진입점에 `경량 테스트 데이터`, `대용량 테스트 데이터` 두 버튼을 노출한다.

### 논의 필요

- 없음

### 선택지

- 기존 대용량 샘플을 편집해서 경량 샘플로 대체
- 대용량 샘플은 유지하고 별도 경량 샘플 파일 추가

### 추천

- 기존 대용량 샘플은 그대로 두고, 별도 경량 샘플 파일을 추가해 사용자가 목적에 맞게 선택하도록 한다.
