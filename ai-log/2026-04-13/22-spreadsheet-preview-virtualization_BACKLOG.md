## 차수 1

### 작업내용

- 좌측 파일 프리뷰의 xlsx/csv 미리보기를 전체 HTML 렌더링 방식에서 행 가상화 기반 그리드로 전환한다.
- 대용량 파일에서도 세로 스크롤 시 동시에 붙는 DOM 수를 줄여 스크롤 렉을 완화한다.
- 기존 헤더/가로 스크롤 경험은 유지하되, 렌더링 방식만 가볍게 교체한다.

### 논의 필요

- 없음

### 선택지

- 전체 HTML 렌더링을 유지하고 CSS만 조정한다.
- 가상화된 그리드 프리뷰로 바꿔 렌더 DOM 수 자체를 줄인다.

### 추천

- 가상화로 바꾼다. 현재 병목은 스타일보다 DOM 수가 너무 많다는 점이라서 CSS 조정만으로는 한계가 있다.
