# Personal Space

개발 계획, backlog, PR 초안, 검토 메모는 이 경로를 기본 작업 공간으로 사용한다.

## 기본 규칙

- backlog 경로: `personal_space/YYYY-MM-DD/N-적당한이름_BACKLOG.md`
- PR 초안 경로 예시: `personal_space/YYYY-MM-DD/N-적당한이름_PR_DRAFT.md`
- 완료한 backlog는 같은 번호를 유지한 채 `(완)`을 붙여 보관

## backlog 최소 항목

- `작업내용`
- `논의 필요`
- `선택지`
- `추천`
- `사용자 방향`
