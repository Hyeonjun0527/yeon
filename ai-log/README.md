# AI Log

이 폴더는 팀원이 AI와 협업하며 남긴 **과정 기록**을 관리하는 공간입니다.

## 목적

- AI와 어떤 문제를 다뤘는지 기록합니다.
- 프롬프트, 결과물, 의사결정 과정을 추적합니다.
- 공식 문서로 승격하기 전의 초안과 사고 과정을 보관합니다.

## 구조

```txt
ai-log/{person}/{yyyy-mm-dd}/...
```

예시:

```txt
ai-log/hyeonjun/2026-04-30/1-작업-codex_....md
ai-log/yeon/2026-04-30/README.md
```

하루 기록이 짧으면 `ai-log/{person}/{yyyy-mm-dd}.md`를 사용할 수 있고, 기록이 길거나 파일이 여러 개면 날짜 폴더를 사용합니다.

## 원칙

1. 결정된 내용은 `docs/`로 승격합니다.
2. 개발 백로그는 `docs/product/backlog/`에 둡니다.
3. AI 대화 원문 전체를 무조건 저장하지 않고 핵심 요약, 결정사항, 후속 작업을 남깁니다.
4. 공식 문서와 중복되면 공식 문서를 기준으로 봅니다.
5. 개인 실험 또는 로컬 임시는 `.local` 또는 gitignore 대상으로 관리합니다.
