# 스페이스 템플릿 복제 및 생성 미리보기 개선 백로그

## 차수 1

### 작업내용

- 기존 기본/사용자 템플릿을 복제해서 새 사용자 템플릿으로 저장하는 흐름을 추가한다.
- 스페이스 설정 drawer에서 선택된 템플릿의 핵심 구조(탭/필드 수) 외에 복제 액션을 제공한다.
- 새 스페이스 생성 모달에서 선택한 템플릿의 구조를 더 분명하게 이해할 수 있도록 미리보기 정보를 강화한다.

### 논의 필요

- 복제 시 이름 suffix를 자동으로 붙일지, 사용자 입력을 받을지 기준이 필요하다.
- 생성 모달 미리보기를 카드 요약 수준으로 유지할지, 일부 탭/필드 샘플까지 노출할지 범위 판단이 필요하다.

### 선택지

1. 템플릿 복제 + 선택 템플릿 구조 요약 미리보기만 먼저 지원한다.
2. 템플릿 복제와 함께 별도 전체 미리보기 모달까지 도입한다.

### 추천

- 선택지 1
- 지금 구조에서는 drawer와 생성 모달 안에서 닫히는 작은 개선이 가장 안전하다. 복제 기능과 요약 미리보기만으로도 실제 사용성은 바로 올라간다.
