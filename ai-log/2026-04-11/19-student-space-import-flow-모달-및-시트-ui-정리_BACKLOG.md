# 19-student-space-import-flow-모달-및-시트-ui-정리

## 작업내용

- 학생관리 화면에서 중복 CTA를 제거한다.
- `스페이스 만들기`를 인라인 생성이 아닌 모달 기반 2갈래 흐름으로 전환한다.
- 모달 선택지는 `빈 스페이스 만들기`, `AI로 파일 가져와 스페이스 만들기`로 구성한다.
- AI 가져오기는 `Google Drive`, `OneDrive`, `내 컴퓨터`를 지원하는 흐름으로 정리한다.
- 기존 Google Sheets 연동 UI는 기능을 유지하면서 차분한 업무툴형 톤으로 재디자인한다.
- 기존 `수강생 데이터 연동하기` / `시트에서 가져오기` 역할은 새 생성 흐름에 통합한다.

## 논의 필요

- 없음. 현재 구현은 이미 사용자와 합의된 방향으로 진행한다.

## 선택지

1. 기존 인라인 생성 유지 + 일부 버튼만 정리
2. 모달 기반 생성 흐름으로 완전 전환
3. 외부 가져오기만 강조하고 시트 UI는 유지

## 추천

- 2번. 부트캠프 운영 맥락상 외부 데이터 기반 생성이 핵심이므로, `스페이스 만들기` 자체를 선택형 모달로 재구성하고 시트 UI도 그 흐름 안에 정돈하는 것이 가장 일관적이다.

## 사용자 방향

- `스페이스 만들기`는 모달로 전환
- 선택지는 `빈 스페이스 만들기` / `AI로 파일 가져와 스페이스 만들기`
- Google Drive / OneDrive / 내 컴퓨터 지원
- Google Sheets 연동 UI는 점잖고 정돈된 업무툴형 디자인으로 개선
- 첫 버전 파일 범위는 엑셀/CSV
