# 클라우드 가져오기 XLSX 미리보기 적층 레이아웃 백로그

## 차수 1

### 작업내용

- `apps/web/src/features/cloud-import/components/cloud-import-inline.tsx`의 현재 좌우 분할 레이아웃을 상하 적층 레이아웃으로 변경한다.
- 상단에는 파일 미리보기(`FilePreview`)를, 하단에는 분석/수정 패널(`ImportRightPanel`)을 배치한다.
- 기존 파일 선택/분석/가져오기 흐름과 버튼 동작은 유지한다.
- 수동 가로 리사이즈 UI와 관련 상태를 제거한다.
- `apps/web/src/server/services/file-analysis-service.ts`에서 의미 없는 커스텀 컬럼 생성을 줄이고, 코호트명을 compact 형식으로 정규화한다.

### 논의 필요

- 데스크톱에서도 완전한 상하 적층을 강제할지, 특정 초대형 화면에서만 추가 변형을 허용할지 결정이 필요하다.

### 선택지

1. 모든 화면에서 상하 적층 고정
2. 기본은 상하 적층, 초대형 화면에서만 추가 레이아웃 확장 검토

### 추천

- 이번 변경은 사용자 요청 그대로 모든 화면에서 상하 적층 고정으로 적용한다. XLSX 열이 많은 현재 문제를 가장 직접적으로 해결하고 구조도 단순해진다.
