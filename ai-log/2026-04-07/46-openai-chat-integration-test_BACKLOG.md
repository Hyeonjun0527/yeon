# 46-openai-chat-integration-test_BACKLOG

## 1차

### 작업내용

- OpenAI Responses API를 사용하는 최소 채팅 연동 서비스와 테스트 스크립트를 추가한다.
- `OPENAI_API_KEY`만 넣으면 모델 호출이 실제로 되는지 바로 확인할 수 있는 검증 커맨드를 만든다.

### 논의 필요

- 연동 테스트를 HTTP route로 열지, 서버 전용 스크립트로만 둘지 여부

### 선택지

- A. 서버 전용 서비스 + CLI 테스트 스크립트로만 두고 외부 노출을 막는다
- B. 별도 테스트 route를 열어 브라우저나 curl로 검증한다

### 추천

- A. 지금 목적은 연동 확인만이므로 가장 작은 범위로 끝내고, OpenAI 키를 쓰는 테스트 경로를 외부 HTTP로 노출하지 않는 편이 안전하다.
