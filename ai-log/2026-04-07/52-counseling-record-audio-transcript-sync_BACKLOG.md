# 52. 상담 원문 뷰어 오디오 동기화

## 작업내용

### 1차

- 저장된 상담 기록 상세 화면에 원본 음성 플레이어를 추가한다.
- 현재 재생 위치를 기준으로 활성 세그먼트를 표시한다.
- 원문 세그먼트를 클릭하면 해당 시점으로 이동하도록 연결한다.
- 오디오 로드 실패 상태를 화면에 분명하게 노출한다.

## 논의 필요

- 자동 재생까지 붙일지, 클릭 시 시킹만 할지 선택이 필요하다.
- 현재는 브라우저 기본 audio UI를 유지하고, 커스텀 파형 UI는 후순위로 둔다.

## 선택지

- 선택지 A: 기본 audio + 세그먼트 클릭 시 이동
- 선택지 B: 커스텀 컨트롤과 파형까지 한 번에 구현

## 추천

- 선택지 A.
- 지금 가장 중요한 것은 원문 신뢰성과 검토 속도이며, 기본 audio 컨트롤만으로도 실제 사용성이 크게 올라간다.
